# **App Name**: FlowMaster

## Core Features:

- Template Designer: Design request templates with definable fields and steps using a drag-and-drop interface.
- Request Submission: Submit new requests based on the created templates, automatically initiating the workflow.
- Workflow Tracking: Track the status and history of each request as it moves through the defined steps.
- Status Visualization: Visualize the current status of each workflow, highlighting pending tasks and completed steps.
- Document Archive: Secure storage for approved/completed requests and the documents attached
- Intelligent Task Assignment: Leverage AI as a tool to predict which user is the optimal candidate to be assigned the next pending step in the workflow.

## Style Guidelines:

- Primary color: Steel blue (#4682B4) for professionalism and trust.
- Background color: Light gray (#E0E0E0) for a clean and modern look.
- Accent color: Electric blue (#7DF9FF) to highlight key actions and status updates.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined look suitable for both headlines and body text
- Use clean, minimalist icons to represent workflow steps and actions.
- A clear, structured layout to visualize process status and actions required for each step.
- Use subtle animations to guide users through the workflow process and provide feedback.
